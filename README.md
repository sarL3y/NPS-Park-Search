# NPS-Park-Search
The user must be able to search for parks in one or more states.<br>
The user must be able to set the max number of results, with a default of 10.<br>
The search must trigger a call to NPS's API.<br>
The parks in the given state must be displayed on the page. Include at least:<br>
Full name<br>
Description<br>
Website URL<br>
The user must be able to make multiple searches and see only the results for the current search.<br>
As a stretch goal, try adding the park's address to the results.<br>
